# Vancouver 311 Service Request Hotspot Analysis

## Project Overview
Spatial hotspot analysis of City of Vancouver 311 service requests 
to identify infrastructure maintenance patterns across Vancouver 
neighbourhoods.

Status: In Progress

## Business Question
Where are the highest concentrations of infrastructure-related 311 
service requests in Vancouver, and what does this reveal about 
maintenance priorities across neighbourhoods?

## Tools & Technologies
- ArcGIS Pro — Kernel Density Estimation, Spatial Joins
- Python (pandas) — Data cleaning and preprocessing
- Power BI — Interactive dashboard and neighbourhood summary
- City of Vancouver Open Data Portal — Source dataset

## Data Source
City of Vancouver Open Data Portal  
Dataset: 3-1-1 Service Requests  
License: Open Government Licence – Vancouver  
URL: https://opendata.vancouver.ca
